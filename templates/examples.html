Source Code Examples

Written by Claus Topholt.

Here a very simple program that outputs 5: 

    script
    {
        output(2 + 3);
    }

Here's a program that counts from 0 to 10: 

    script
    {
        # Use a variable to count.
        a = 0;
        while (a < 10)
        {
            a = a + 1;
            output(a);
        }
    }

Here's a program that uses if-else statements and some convoluted math to show off parentheses: 

    script
    {
        a = 5;
        b = 10;
        if (a * 3 - (2 * 2) > b)
        {
            output(a);
        }
        else
        {
            output(b);
        }
    }

Here's an infinite loop. The virtual machine will automatically stop after 10.000 instructions have been executed: 

    script
    {
        a = 0;
        while (1)
        {
            a = a + 1;
        }
    }
Tip: Try to refresh the browser as the program runs.
